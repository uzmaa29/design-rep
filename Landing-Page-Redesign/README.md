# Savana Website – UI/UX Redesign (Mobile → Desktop)

## Overview

This project is a UI/UX redesign of the Savana e-commerce interface, where the original mobile-first layout was reimagined into a structured and user-friendly desktop experience.

The redesign focuses on improving usability, reducing visual clutter, and creating a more intuitive shopping journey.

---

## Problem Statement

The original interface (mobile version) had:

* Weak visual hierarchy
* Limited navigation clarity
* Poor content distribution for larger screens

This resulted in a distracting and less efficient user experience.

---

## Design Goals

* Create a clean and structured desktop layout
* Improve navigation and product discoverability
* Establish strong visual hierarchy
* Balance promotional content with usability
* Enhance overall user flow

---

## Process

### 1. Analysis (Before UI)

* Studied layout patterns and user flow
* Identified clutter, excessive banners, and lack of spacing
* Noted absence of desktop optimization

### 2. Wireframing

* Designed low-fidelity layouts for desktop
* Focused on grid structure and spacing
* Organized sections: Hero → Categories → Products → Footer

### 3. Final UI Design

* Built high-fidelity screens in Figma
* Introduced consistent spacing and alignment
* Used clear typography and hierarchy
* Reduced unnecessary visual noise

---

## Key Design Improvements

### 1. Hero Section Redesign

* Replaced cluttered promotional banners with a **focused split layout**
* Clear headline + CTA (“Shop Now”) improves conversion intent

### 2. Navigation Enhancement

* Converted compact mobile menu into a **full horizontal navigation bar**
* Added clear category labels (Sales, Clothing, Accessories, etc.)

### 3. Visual Hierarchy

* Structured content into sections with breathing space
* Reduced competing elements to guide user attention

### 4. Product Discovery

* Introduced **Hot Categories** for quick browsing
* Cleaner product grid improves scanning

### 5. Information Clarity

* Service highlights (returns, delivery, shipping) made more visible
* Footer organized into clear sections

---

## Before vs After

### Before

* Cluttered UI with excessive promotional elements
* Mobile layout stretched into larger view
* Weak hierarchy and navigation

### After

* Clean, structured desktop layout
* Clear user flow from entry to product exploration
* Balanced visuals and functionality

## Key Learnings

* Designing for desktop requires **layout restructuring, not scaling**
* Visual hierarchy directly impacts usability
* Less clutter = better user decisions
* Grid systems improve consistency and alignment
